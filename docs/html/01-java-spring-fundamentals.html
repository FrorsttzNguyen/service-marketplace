Java & Spring Boot Fundamentals

Tai lieu nay giai thich tat ca cac khai niem Java va Spring Boot can thiet de hieu duoc code trong du an Service Marketplace. Moi khai niem deu duoc lien ket voi code thuc te trong project.

1. Java la gi?

JVM, JDK, JRE

Khai niem Viet tat Giai thich don gian
JVM Java Virtual Machine May ao chay Java bytecode. Day la ly do Java co the chay tren moi he dieu hanh — "Write once, run anywhere". Khi Hien compile .java thanh .class, JVM doc bytecode va thuc thi.
JRE Java Runtime Environment Moi truong de chay Java application. Gom JVM + thu vien co ban. Du de chay app, nhung khong du de phat trien.
JDK Java Development Kit Bo cong cu phat trien. Gom JRE + compiler (javac) + cac tool khac. Day la thu Hien cai dat de code Java. Project nay dung JDK 21.

Tai sao chon Java cho backend?

  • Strong typing — Loi type bi bat tai compile-time, khong phai luc runtime. Giam bug.
  • Ecosystem lon — Spring Boot, Hibernate, Maven, lib cho moi thu da co san va kien dinh.
  • Enterprise standard — Phan lon cong ty lon dung Java backend. Hoc Spring Boot = co hieu nhieu viec.
  • Performance — JVM toi uu code luc runtime (JIT compilation), nhanh hon nhieu so voi interpreter thuan.

Java 21 — cac tinh nang dung trong project

Project nay dung Java 21 (xem pom.xml: <java.version>21</java.version>). Mot so tinh nang moi se gap:

  • Records (Java 16+) — Cach tao data class bat bien (immutable) ngan gon. Se gap khi dinh nghia DTO.
  • Sealed classes (Java 17+) — Gioi han class nao duoc ke thua. Co the dung cho domain events.
  • Pattern matching for switch (Java 21) — Viet ngan hon khi kiem tra type.

Tip

Khong can hieu ngay tat ca tinh nang moi. Khi gap trong code, tai lieu nay se giai thich.

2. Java Basics dung trong Project

Classes & Objects

Class la ban thiet ke (blueprint). Object la the hien cu the cua class do. Moi file Java deu co it nhat mot class.

Vi du tu project — file ServiceMarketplaceApplication.java:

src/main/java/com/hien/marketplace/ServiceMarketplaceApplication.java 
// package = thu muc chua file nay. Tat ca class trong project nam trong com.hien.marketplace
package com.hien.marketplace;

// import = lay class tu package khac de dung
import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication  // ← Annotation: bao cho Spring biet day la main app class
public class ServiceMarketplaceApplication {

    // public static void main = diem bat dau (entry point) cua moi Java application
    // public  = bat ky ai cung co the goi method nay
    // static  = khong can tao object de goi method nay
    // void    = method khong tra ve gia tri
    // String[] args = tham so dong lenh (command-line arguments)
    public static void void main(String[] args) {
        // SpringApplication.run() = khoi dong Spring Boot application
        SpringApplication.run(ServiceMarketplaceApplication.class, args);
    }
}

Enums

Enum (enumeration) la mot cach dinh nghia tap hop cac gia tri co dinh. Trong project se dung de dinh nghia vai tro nguoi dung:

public enum UserRole {
    CUSTOMER,   // Nguoi dat dich vu
    VENDOR,     // Nguoi cung cap dich vu
    ADMIN       // Quan tri vien
}

// Cach dung:
UserRole role = UserRole.CUSTOMER;

// Kiem tra gia tri:
if (role == UserRole.VENDOR) {
    // hien thi dashboard cua vendor
}

Note

Enum khac voi String o cho: chi nhan dung cac gia tri da dinh nghia. Compiler se bat loi neu go sai. Dung String thi co the go sai ma khong ai bao den luc runtime.

Annotations (Phu chu / Chu thich)

Annotation la mot cach them thong tin vao code, ma Java hoac framework (nhu Spring) co the doc va xu ly. Luon bat dau bang dau @.

Annotation Ai xu ly? Y nghia
@Override Java compiler Bao rang method nay ghi de method cua class cha. Neu ghi sai ten, compiler bao loi.
@Entity Hibernate/JPA Class nay tuong ung voi mot bang trong database.
@RestController Spring Class nay xu ly HTTP request va tra ve JSON.
@Bean Spring Object tra ve tu method nay se duoc Spring quan ly.
@Configuration Spring Class nay chua cac dinh nghia bean (cac @Bean method).

Generics (Kieu tong quat)

Generics cho phep viet code hoat dong voi nhieu kieu du lieu khac nhau, nhung van dam bao type safety tai compile-time.

// Khong dung generics — khong biet trong List co gi, phai ep type (cast)
List users = new ArrayList();
users.add("hello");  // String? User? Khong ai kiem tra!

// Dung generics — compiler dam bao chi chua User
List<User> users = new ArrayList<>();
users.add(new User());  // OK
// users.add("hello");  // Compile error!

// Optional <T> = co the co hoac khong co gia tri
Optional<Booking> booking = bookingRepository.findById(1L);
// booking co the chua Booking hoac chua nothing (empty)
// Tranh duoc NullPointerException

Can than

Optional khong phai la cach tranh null hoan toan. Chi dung Optional cho kieu tra ve cua method, khong dung cho field cua class hoac parameter cua method.

Records (Java 16+)

Record la cach tao data carrier class ngan gon. Tu dong tao constructor, getter, equals, hashCode, toString.

// Truoc Java 16: phai viet nhieu boilerplate
public class UserDto {
    private final String name;
    private final String email;
    // + constructor + getters + equals + hashCode + toString...
}

// Java 16+: dung record, 1 dong
public record UserDto(String name, String email) {}

// Cach dung:
UserDto dto = new UserDto("Hien", "hien@example.com");
dto.name();  // "Hien" (khong phai getName(), ma la name())

Records la immutable — khong the thay doi gia tri sau khi tao. Dieu nay an toan cho multi-threading va don gian cho reasoning.

3. Spring Framework la gi?

Van de khi khong co Spring

Trong Java thuan, Hien phai tu tao va quan ly moi object:

// Khong co Spring: phai tu tao moi thu
public class BookingController {
    // Phai tu tao dependency — tight coupling!
    private BookingService bookingService = new BookingService();
    private BookingRepository repo = new BookingRepository();
}

// Van de:
// 1. BookingService duoc tao o day, thay doi BookingService anh huong controller
// 2. Khong de test (khong the mock dependency)
// 3. Moi class phai tu quan ly vong doi cua dependency

Inversion of Control (IoC) — Dao nguoc dieu khien

Thay vi code cua Hien tao object, Spring tao va quan ly object. Object duoc Spring quan ly goi la Bean.

Dependency Injection (DI) — Thay vi class tu tao dependency, Spring inject (truyen vao) dependency do.

// Co Spring: Spring inject dependency qua constructor
@RestController
public class BookingController {
    private final BookingService bookingService;

    // Spring tu dong truyen BookingService vao (inject)
    // Khong can = new BookingService() nua!
    public BookingController(BookingService bookingService) {
        this.bookingService = bookingService;
    }
}

Cac annotation quan trong

Annotation Vai tro Giai thich
@Component Bean co ban Danh dau class la bean. Spring scan va dang ky tu dong.
@Service Business logic Giong @Component nhung the hien rang class nay chua logic nghiep vu. De doc hon.
@Repository Data access Danh dau class truy cap database. Tu dong chuyen exception cua DB thanh Spring exception.
@Controller Web layer Xu ly HTTP request, tra ve HTML view. It dung trong REST API.
@RestController REST API layer = @Controller + @ResponseBody. Tra ve JSON/XML thay vi HTML.
@Configuration Config Class chua cac @Bean method dinh nghia bean.
@Bean Tao bean Method nay tra ve mot object duoc Spring quan ly.

Tip

@Service, @Repository, @Controller deu la @Component. Ve mat ky thuat chung giong nhau. Su dung dung annotation giup code doc duoc de hieu hon — thay @Service la biet class nay chua business logic.

4. Spring Boot Magic

Spring Boot them gi vao Spring?

Spring Framework cung co IoC va DI, nhung cau hinh rat nhieu. Spring Boot them 3 dieu chinh:

  • Auto-configuration — Tu dong cau hinh dua tren dependencies co trong classpath. Co JPA? Tu cau hinh datasource. Co Security? Tu bat security.
  • Embedded server — Tomcat duoc built-in. Khong can cai dat server rieng. Chay main() la duoc web server.
  • Starter dependencies — Mot dependency keo theo tat ca thu can thiet. Vi du spring-boot-starter-web bao gom: Spring MVC + Tomcat + Jackson (JSON).

5. Maven la gi?

Maven — cong cu quan ly du an

Maven la build tool cho Java. No lam 3 viec chinh:

  1. Quan ly dependencies — Tu tai ve cac thu vien can thiet (JAR files).
  2. Build — Compile code, chay test, dong goi thanh JAR/WAR.
  3. Chuan hoa cau truc project — Moi project Maven co cung cau truc thu muc.

pom.xml — Project Object Model

File pom.xml nam o goc project. No mo ta toan bo project. Phan tich tung phan:

6. Code Walkthrough: SecurityConfig.java

Phan tich tung dong trong file cau hinh bao mat cua project.

src/main/java/com/hien/marketplace/config/SecurityConfig.java 
package com.hien.marketplace.config;

// import: lay cac class can dung tu Spring Security
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.security.config.annotation.web.builders.HttpSecurity;
import org.springframework.security.config.annotation.web.configuration.EnableWebSecurity;
import org.springframework.security.config.annotation.web.configurers.AbstractHttpConfigurer;
import org.springframework.security.config.http.SessionCreationPolicy;
import org.springframework.security.web.SecurityFilterChain;

@Configuration   // Bao Spring: class nay dinh nghia beans
@EnableWebSecurity  // Bat Spring Security features
public class SecurityConfig {

    @Bean  // Object tra ve duoc Spring quan ly (bean)
    public SecurityFilterChain securityFilterChain(HttpSecurity http) throws Exception {
        http
            // Quan ly session: STATELESS = khong tao HTTP session
            // Nghia la moi request phai tu chung thuc (JWT token)
            .sessionManagement(session ->
                session.sessionCreationPolicy(SessionCreationPolicy.STATELESS)
            )
            // CSRF (Cross-Site Request Forgery): Tat vi day la REST API
            // CSRF bao ve form submission trong browser. REST API dung JWT, khong can.
            .csrf(AbstractHttpConfigurer::disable)
            // Cho phep tat ca request (Phase 0). Phase 2 se gioi han theo role.
            .authorizeHttpRequests(auth ->
                auth.anyRequest().permitAll()
            );

        return http.build();  // Xay dung va tra ve SecurityFilterChain
    }
}

Tai sao tat CSRF?

CSRF la tan cong loi dung browser tu dong gui request den trang web da xac thuc. Nhung tan cong nay chi hoat dong khi dung cookie-based session.

Project nay dung JWT token trong Authorization header, khong dung cookie de xac thuc. Vi vay CSRF khong la moi de va nen tat de API don gian hon.

STATELESS session nghia la gi?

Mac dinh Spring Security tao HttpSession de luu thong tin dang nhap. Khi set STATELESS, Spring khong tao session. Moi request phai mang thong tin xac thuc rieng (JWT token trong header).

Note — Phase 0

permitAll() cho phep moi nguoi goi bat ky endpoint nao ma khong can dang nhap. Day la cau hinh tam thoi. Khi den Phase 2, se thay bang phan quyen theo role (CUSTOMER, VENDOR, ADMIN).

7. Code Walkthrough: HealthController.java

src/main/java/com/hien/marketplace/interfaces/rest/HealthController.java 
package com.hien.marketplace.interfaces.rest;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;
import java.util.Map;

@RestController  // = @Controller + @ResponseBody
// @Controller: class nay xu ly HTTP request
// @ResponseBody: gia tri tra ve duoc ghi truc tiep vao response body (JSON)
// Khi ket hop lai: class nay nhan HTTP request va tra ve JSON
public class HealthController {

    @GetMapping("/api/health")  // Map HTTP GET /api/health → method health()
    public Map<String, String> health() {
        // Map.of() = tao immutable map (khong the thay doi sau khi tao)
        // Java tu dong chuyen Map → JSON nho Jackson library
        return Map.of(
            "status", "UP",
            "service", "service-marketplace"
        );
    }
}

Giai thich tung buoc:

  1. Browser gui HTTP GET request den /api/health.
  2. Tomcat (embedded web server) nhan request.
  3. DispatcherServlet (Spring MVC) tim xem method nao map voi URL nay. Tim thay @GetMapping("/api/health").
  4. HealthController.health() duoc goi. Tra ve Map.of("status", "UP", "service", "service-marketplace").
  5. Jackson (JSON library) chuyen Map thanh JSON string.
  6. Response tra ve browser voi status 200 va body JSON.
